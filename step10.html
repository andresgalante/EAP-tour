Control user access

Add and assign roles to users and groups. Define custom permissions for roles and view role membership.

By default, WildFly uses simple access controls for the management interfaces. Optionally, you can choose to implement Role-Based Access Control (RBAC) to specify a set of permissions for management users. The roles assigned to a user or group determine which resources can be accessed and which operations can be performed.

Manage the list of roles assigned to a particular user or group.
